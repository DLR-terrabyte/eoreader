eoreader.bands.alias module¶

Aliases for bands and index, created in order to import just this file and not OpticalBandNames, SarBandNames and index.

To use it, simply type:

`python >>> from eoreader.bands.alias import * >>> GREEN <OpticalBandNames.GREEN: 'GREEN'> >>> HH <SarBandNames.HH: 'HH'> >>> NDVI <function NDVI at 0x00000261F6FFA950> `
